# My Invoice Generator

A clean and simple web-based invoice generator built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui.

## Features
- Create professional invoices
- Live preview
- Export as PDF
- Save invoices locally
- Send via email (optional)
- Multiple templates

## Getting Started

1. Clone or download the project
2. Run `npm install`
3. Create `.env.local` (for email feature – optional)
4. Run `npm run dev`

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Hook Form + Zod
- Puppeteer (PDF generation)
